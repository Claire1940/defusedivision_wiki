<Section icon={Shield} title="Sicherheit, Moderation und Scam-Checks">

Discord ist nützlich, aber offene Server ziehen auch Fake-Gewinnspiele, Phishing-Links und Account-Fallen an. Wenn eine Nachricht dringend oder zu großzügig klingt, verlangsame dein Tempo. Die meisten Scams setzen auf Druck, nicht auf Logik. Eine gute Regel ist einfach: Keine unbekannten Dateien anklicken, keine Zugangsdaten teilen und keinen inoffiziellen „Belohnungs“-Seiten vertrauen.

Auch Moderation ist wichtig. Bleib höflich, vermeide es, Kanäle zu fluten, und melde Regelverstöße über den richtigen Weg. So bleibt der Server für alle lesbar und die Moderatoren können ihre Zeit auf echte Probleme statt auf Aufräumarbeit verwenden.

<Callout type="warning" title="Warnsignale, auf die du achten solltest">
Jede Nachricht, die nach deinem Login, deiner Browser-Session, deinem Token oder deinem Passwort fragt, sollte als unsicher gelten. Seriöse Moderatoren brauchen keine privaten Zugangsdaten.
</Callout>

| Warnsignal | Warum es riskant ist | Sichere Reaktion |
|---|---|---|
| „Free reward“-DM | Häufige Phishing-Falle | Ignorieren, blockieren, melden |
| Fake-Verifizierungslink | Kann Kontozugriff stehlen | Nur über vertrauenswürdige Kanäle verifizieren |
| Unvertrauenswürdiger Download | Mögliche Malware | Nicht installieren |
| Druck, schnell zu handeln | Nimmt dir das Zeitfenster zum Nachdenken | Pause machen und bestätigen |
| Nachfrage nach deinem Passwort | Für Support nie nötig | Sofort ablehnen |
