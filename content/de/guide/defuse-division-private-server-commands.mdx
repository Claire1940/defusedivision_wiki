<Section icon={Swords} title="Essenzielle Admin- und Spielerbefehle">

Die Verwaltung eines Servers erfordert ein Gleichgewicht zwischen Utility und Moderation. Die folgenden **Befehle für private Server in Entschärfungs-Division** werden am häufigsten von Administratoren verwendet, um die Ordnung aufrechtzuerhalten und spezifische Gameplay-Szenarien zu ermöglichen. Diese Befehle werden normalerweise direkt in den Spiel-Chat eingegeben.

| Befehl | Syntax | Effekt |
| :--- | :--- | :--- |
| **Fly** | :fly [Spieler] | Aktiviert das Fliegen für den Zielspieler |
| **Unfly** | :unfly [Spieler] | Deaktiviert das Fliegen für den Zielspieler |
| **Speed** | :speed [Spieler] [Zahl] | Legt die Laufgeschwindigkeit eines Spielers fest |
| **JumpPower** | :jp [Spieler] [Zahl] | Ändert die Sprunghöhe |
| **Teleport** | :tp [P1] [P2] | Teleportiert Spieler 1 zu Spieler 2 |
| **Bring** | :bring [Spieler] | Teleportiert einen Spieler zu Ihrem Standort |

Über einfache Bewegungen hinaus sind Moderationsbefehle entscheidend, um Ihren privaten Bereich vor „Trollen“ oder ungebetenen Gästen zu schützen, die eventuell Ihren Link gefunden haben.

| Moderation | Syntax | Dauer |
| :--- | :--- | :--- |
| **Kick** | :kick [Spieler] [Grund] | Sofortige Entfernung |
| **Ban** | :ban [Spieler] [Grund] | Permanente Entfernung vom Server |
| **Mute** | :mute [Spieler] | Deaktiviert den Spieler-Chat |
| **Unmute** | :unmute [Spieler] | Aktiviert den Spieler-Chat |

<Callout type="warning" title="Warnung">
Seien Sie vorsichtig bei der Verwendung des `:ban`-Befehls. Auf privaten Servern werden Bans oft im Datenspeicher des Servers hinterlegt und können über verschiedene Sitzungen hinweg bestehen bleiben, sofern sie nicht manuell aufgehoben werden.
</Callout>

</Section>

<Section icon={Sparkles} title="Taktische Übungsbefehle">

Für Spieler, die ihre Fähigkeiten verbessern wollen, bieten **Befehle für private Server in Entschärfungs-Division** spezialisierte Werkzeuge, die reale Spielszenarien simulieren. Diese sind besonders nützlich, um „Lineups“ (spezifische Punkte zum Werfen von Granaten) zu lernen oder „Pre-Fire“-Winkel zu testen.

| Taktisches Ziel | Vorgeschlagener Befehl | Praktische Anwendung |
| :--- | :--- | :--- |
| **Granatentraining** | :viewlineups / :infammo | Flugbahn sehen und Würfe üben |
| **Positionierung** | :setpos [x,y,z] | Zu Testzwecken an exakt dieselbe Stelle zurückkehren |
| **Timing** | :timer [Sekunden] | Testen, wie lange eine Rotation dauert |
| **Sichtbarkeit** | :esp [on/off] | Spielerumrisse durch Wände sehen (nur Admin) |

Diese Befehle verwandeln das Spiel von einem Standard-Shooter in ein Labor für taktische Exzellenz. Durch das Beherrschen der Befehle `:setpos` und `:bring` kann ein Coach ein ganzes Team sofort über die Karte bewegen, um eine spezifische Strategie zu demonstrieren.

Verwenden Sie den Befehl `:bind` (sofern vom Admin-Skript unterstützt), um häufige Befehle wie `:respawn` oder `:fly` auf bestimmte Tasten Ihrer Tastatur zu legen, um schneller darauf zugreifen zu können.
