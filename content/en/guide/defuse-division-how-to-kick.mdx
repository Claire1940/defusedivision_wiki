<Section icon={Target} title="Defuse Division How to Kick: Step-by-Step Tutorial">

Initiating a kick requires using the correct syntax in the game chat. Because the game relies on Roblox's standard chat filter and user configuration, entering the command precisely is necessary for the system to register the vote.

Follow these steps to start a vote kick against a disruptive player in your match:

<Steps>
  <Step num="1" title="Open the In-Game Chat">
    Press the "/" key on your keyboard to open the active text chat window during a round or during the buy phase.
  </Step>
  <Step num="2" title="Enter the Kick Command">
    Type the command followed by the target player's name. Use the format: `/votekick [PlayerName]`. Make sure to use their actual username rather than their display name if they differ.
  </Step>
  <Step num="3" title="Submit the Vote Request">
    Press Enter to broadcast the vote to your team. A prompt will appear on the screens of your teammates, allowing them to cast their vote.
  </Step>
  <Step num="4" title="Await the Final Verdict">
    Monitor the voting progress. The target player will be removed from the server once the required majority threshold is reached.
  </Step>
</Steps>

The voting window remains open for a limited time. If the threshold is not met before the timer expires, the vote fails, and a cooldown is applied to prevent spamming.

| Feature | Details | Status |
| :--- | :--- | :--- |
| **Command Format** | /votekick [username] | Active |
| **Required Votes** | Majority of active team | Required |
| **Cooldown Period** | 120 seconds between votes | Active |
| **Minimum Players** | At least 4 players in server | Required |

Using the system requires coordination with the rest of your squad. If you do not explain why you are starting the vote, your teammates may ignore the prompt or vote against it.

<Callout type="tip" title="Username Precision">
Ensure you type the player's exact display name or username to prevent the command from failing.
</Callout>

</Section>

<Section icon={Swords} title="Vote Kick Rules and Etiquette">

In competitive tactical shooters, emotions run high. However, the vote kick tool should only be used for legitimate disruptions. Sabotaging matches, going away from the keyboard (AFK) for multiple rounds, or using third-party software are all valid reasons to remove a player. 

Conversely, kicking a player simply because they have a lower skill level or failed to win a round is considered poor sportsmanship. This behavior reduces the player base and creates a toxic environment.

| Scenario | Justification | Action Recommended |
| :--- | :--- | :--- |
| **AFK for multiple rounds** | Justified | Initiate vote kick |
| **Active exploiting or cheating** | Justified | Kick and report to Roblox |
| **Failing a 1v5 clutch attempt** | Unjustified | Do not kick / Support teammate |
| **Intentionally team blinding** | Justified | Initiate vote kick |
| **Low skill or learning the game** | Unjustified | Offer constructive tips |

If you find yourself in a lobby where players are attempting to kick teammates for minor mistakes, it is often best to vote against the kick and encourage communication. A team that works together to help struggling players perform better is more likely to win than one that constantly thins its own ranks.

<Callout type="success" title="Positive Reinforcement">
Helping new players learn the map layouts and movement mechanics improves the overall community health.
</Callout>

</Section>
